Auto registration of services

All auto-detected services that implement ContainerAwareInterface will receive the service container automatically.
Any auto-generated services can be overrided by simply defining a service with the same id.

Security voters

{% highlight bash %} src └── App └── Security   └── BlogPostVoter.php {% endhighlight %}
{% highlight bash %} {% include standard/auto-registration/security.yml %} {% endhighlight %}
{% highlight bash %} The service 'app.security.blog_post_voter' will be automatically created. The service container will be injected in 'app.security.blog_post_voter'. {% endhighlight %}
